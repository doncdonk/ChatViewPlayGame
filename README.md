# 🎮 ChatViewPlayGame

**Twitch チャット連動ゲームランチャー（配信者向け）**

配信者がアプリを起動してOBSでキャプチャするだけで、視聴者がチャットで参加できるゲームが楽しめます。

---

## 収録ゲーム

| ゲーム | 概要 |
|--------|------|
| 💣 ViewBomb | チャットコメントでマスを開くマインスイーパー |
| ♟ チャットvsリバーシ | 配信者 vs チャット全体でリバーシ対決 |
| 🏇 競馬 | 馬に投票してポイントを競う・馬育成・グレード設定 |
| 🎨 ペイント | お絵描き配信・チャットが弾幕で流れる |

---

## 起動方法

### exe を使う場合（推奨）

`build.bat` をダブルクリック → `dist\ChatViewPlayGame.exe` が生成されます。

**リリース ZIP の構成：**
```
ChatViewPlayGame_release.zip
├── ChatViewPlayGame.exe
├── token_guide.html       ← Twitch トークン取得ガイド
├── GUIDE.md               ← エンドユーザー向けガイド
└── img/                   ← 競馬用画像（必須）
```

### Python で直接実行

```bash
pip install pillow numpy
python game.py
```

## 初回セットアップ

### Twitch トークンの取得

`token_guide.html` をブラウザで開いてください。

1. https://twitchtokengenerator.com/ にアクセス
2. 「Bot Chat Token」をクリック
3. Twitch にログインして認証
4. ACCESS TOKEN をコピー
5. アプリ → Settings → Token に `oauth:xxxxxxx` の形式で入力

### アプリの設定

Settings 画面で以下を設定：

| 項目 | 内容 |
|------|------|
| Window Size | 1280×720 / **1600×900**（デフォルト）/ 1920×1080 |
| Theme | 7種類から選択 |
| Twitch Channel Name | あなたのチャンネル名（小文字） |
| OAuth Token | `oauth:` から始まるトークン |

### OBS 設定

ソース → `+` → **ウィンドウキャプチャ** → ChatViewPlayGame を選択

## 視聴者コマンド

### 💣 ViewBomb

| コマンド | 動作 |
|----------|------|
| `A1` | A列1行目を開く |
| `flag A1` | A列1行目にフラグを立てる |

### ♟ リバーシ

チャット側のターン中に `A1` `C3` などのマス座標を送ると投票になります。

### 🏇 競馬

| コマンド | 動作 |
|----------|------|
| `1` 〜 `8` | その番号の馬に投票（1人1票） |

## 🏇 競馬機能詳細

### レースグレード

ロビーで5段階のグレードを選択できます。ランダム馬の強さが変わります。

| グレード | ランダム馬の強さ | 騎手技術 | 難易度 |
|----------|-----------------|---------|--------|
| 未勝利 | 1〜3 | 1〜2 | 入門 |
| G3 | 1〜5 | 1〜3 | 標準（デフォルト） |
| G2 | 3〜7 | 2〜4 | やや難しい |
| G1 | 5〜9 | 3〜5 | 難しい |
| 特別G1 | 7〜10 | 4〜5 | 最高峰 |

グレードはレース連続開催時も引き継がれます。

### 馬の育成

競馬ロビーから「🐴 馬を育てる」を選ぶと育成モードに入ります。

1. **馬名を入力**（カタカナ・最大8文字）
2. **12ヶ月の育成**（毎月3択のメニューから選択）
   - メニュー選択後に馬画像アニメーションで演出
   - 調教師のコメント・ランダムイベントあり（怪我・覚醒・好調）
3. **育成完了**でコードが発行される → コピーして保存
4. コードをコピーして保存しておく

**特性システム（23種 + コンボ8種）:**

| カテゴリ | 内容 |
|----------|------|
| 能力値系 | 快速型・鉄人・鬼神・瞬足・不屈 など |
| コース系 | 芝巧者・ダート巧者・長距離適性・短距離砲 など |
| 育成ルート系 | 鍛錬の申し子・無敗の気質・バランス調教 など |
| コンボ特性 | 先行逃げ切り・末脚怪物・天才肌・孤高の怪物 など |

育成終盤の状態がレース当日の調子確率に影響します（最終月に放牧→好調になりやすい など）。

### コードで参戦

「📋 コードで参戦」から最大8頭の育成馬コードを入力してレースに出走できます。空欄はランダム馬で補完されます。

- 各入力欄の「詳細」ボタンで馬のプロフィールをモーダル表示
- 現在のグレード・投票時間・レース数などの設定が画面上部に表示される

### 馬プロフィールモーダル

投票画面・コード参戦画面の「詳」「詳細」ボタンで表示。

- 五角形レーダーチャート・ステータス星・馬場適性・調子・オッズ・騎手情報・特性を表示
- ヘッダー部分をドラッグして移動可能
- × ボタンまたはモーダル外クリックで閉じる

### ロビー設定一覧

| 設定項目 | 内容 | デフォルト |
|----------|------|-----------|
| 投票時間 | 30 / 60 / 90 / 120秒 | 60秒 |
| レース数 | 1 / 3 / 5 / 10 / 無限 | 3 |
| 自動遷移 | 結果後の自動次レース移行時間 | なし |
| グレード | 未勝利 / G3 / G2 / G1 / 特別G1 | G3 |
| 育成馬の連続出走 | 有効/無効 | 無効 |

## 注意事項

- `config.json` にはトークンが含まれます。GitHub に公開しないでください（.gitignore で除外済み）
- トークンは約60日で期限切れになります。期限切れ時は再取得してください
- exe 実行時は `ChatViewPlayGame.exe` と同じフォルダに `img/` が必要です
- ペイントで保存した画像（`paint_*.png`）は exe と同じフォルダに保存されます
- 馬コードには育成馬のステータスは含まれず seed+育成履歴から毎回再計算されます（改ざん防止）
