# 🎮 ChatViewPlayGame

Twitch チャット連動ゲームランチャー（配信者向け）
Python 標準ライブラリのみで動作。追加インストール不要でゲームをプレイできます。

---

## 収録ゲーム

| ゲーム | 状態 | 概要 |
|--------|------|------|
| 💣 ViewBomb | ✅ プレイ可能 | チャットコメントで地雷を避けるマインスイーパー |
| ♟ チャットvsリバーシ | ✅ プレイ可能 | 配信者（黒）vs チャット全体（白）でリバーシ対決 |
| 🏇 競馬 | ✅ プレイ可能 | チャットでレースに投票して参加 |

---

## 起動方法

### Python で直接実行

```bash
python game.py
```

### Windows exe を使う場合

`build.bat` をダブルクリックするだけ。自動で以下を実行します。

1. pip install pillow pyinstaller
2. python build_icon.py  →  icon.ico 生成
3. pyinstaller ChatViewPlayGame.spec  →  exe ビルド
4. dist\token_guide.html をコピー
5. ChatViewPlayGame_release.zip を生成（exe + ガイド）

## 初回セットアップ

### 1. Twitch OAuth トークンの取得

詳しい手順は `token_guide.html` をブラウザで開いてください。

簡易手順:
1. https://twitchtokengenerator.com/ を開く
2. 「Bot Chat Token」をクリック
3. Twitch にログインして認証
4. ACCESS TOKEN をコピー
5. アプリ起動 → 「⚙ Settings」でトークンを入力（先頭に `oauth:` を付ける）

### 2. アプリの設定

「⚙ Settings」画面で以下を設定して保存:

| 項目 | 内容 |
|------|------|
| Window Size | 1280×720 / 1600×900（デフォルト）/ 1920×1080 |
| Theme | 7種類から選択（ラジオボタンでリアルタイムプレビュー） |
| Twitch Channel Name | あなたのチャンネル名（小文字） |
| OAuth Token | `oauth:` から始まるトークン |

設定は `config.json` に自動保存されます（次回起動時は入力不要）。
ウィンドウサイズを変更した場合は再起動が必要です（保存後に再起動ボタンが表示されます）。

## 視聴者コマンド

### 💣 ViewBomb

| コマンド | 動作 |
|----------|------|
| `A1` | A列1行目のセルを開く |
| `flag A1` | A列1行目にフラグを立てる / 外す |

- 列はA〜Z（最大26列）、行は数字
- 配信者はマウスクリックでも操作可能（左クリック: 開く、右クリック: フラグ）
- サイドバーの「📋 Copy hint for chat」でコマンド説明をクリップボードにコピー
- 座標文字サイズはロビー画面のスライダーで調整可能

#### モード
| モード | 内容 |
|--------|------|
| 通常 | 1ゲームずつプレイ |
| 連続 | クリアするたびに地雷が3個増える |

---

### ♟ チャットvsリバーシ

| コマンド | 動作 |
|----------|------|
| `A1` `C3` など | チャット側のターン中に置きたいマスに投票 |

- 配信者（黒）はマウスクリックで石を置く
- チャット（白）のターンは投票を集計して最多票のマスに自動で打つ
- 配信者は `Space` キーで投票を早期締め切り可能
- 最大投票時間を超えると自動締め切り
- 盤面外周に A〜H / 1〜8 の座標ラベルを表示
- チャット側のターン中は合法手マスの座標を白・太字で強調表示

---

### 🏇 競馬

| コマンド | 動作 |
|----------|------|
| `1` 〜 `8` | 該当番号の馬に投票（1人1票） |

- 配信者は投票画面の馬一覧をクリックしても投票可能
- ランダムで生成されたレース名・馬名・コース（芝/ダート）・距離でレースを開催
- 馬のステータス（速度・スタミナ・コーナー・馬場適性・調子）がレース結果に影響
- 投票画面に一言評価を表示（投票の参考に）
- レース距離に応じてレース時間が変化（短距離は速く、長距離はじっくり展開）
- `Space` キーで投票を早期締め切り可能

#### ロビー設定
| 設定 | 選択肢 |
|------|--------|
| 投票時間 | 30 / 60 / 90 / 120秒 |
| レース数 | 1 / 3 / 5 / 10 / 無限 |
| 自動遷移 | なし / 15 / 30 / 60秒 |

#### レース画面フロー
```
投票画面 → 出走演出（ゲートオープン）→ レース中 → ゴール演出 → 結果画面
```

- 結果画面に各馬の投票者名を表示
- 2レース目以降は累積的中ランキングを表示
- 指定レース数完了後に最終ランキング画面を表示

## OBS 設定

1. OBS を起動
2. ソース → `+` → **ウィンドウキャプチャ** を追加
3. ウィンドウ一覧から **ChatViewPlayGame** を選択
4. OK をクリック

ウィンドウサイズは設定で固定されているためキャプチャ枠がズレません。

## 注意事項

- `config.json` にはトークンが含まれます。GitHub などに公開しないでください（`.gitignore` で除外済み）
- トークンは約 60 日で期限切れになります。期限切れの場合は再取得して設定し直してください
- exe 実行時は `ChatViewPlayGame.exe` と同じフォルダに `config.json` が生成されます
