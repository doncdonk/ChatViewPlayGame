SETUP GUIDE

Twitch OAuth トークン
取得ガイド

ChatViewPlayGame を Twitch チャットに接続するために必要なトークンを取得します。所要時間 約2〜3分。

1

サイトにアクセスする

以下のURLをブラウザで開いてください。Twitch公式が認可したトークン生成ツールです。

🔒 twitchtokengenerator.com
✦ Twitch Token Generator by swiftyspiffy

I am here to get a...

🤖
Bot Chat Token チャット読み取り用トークン（こちらを選ぶ）
⚙️
Custom Scope Token カスタムスコープ
「Bot Chat Token」をクリック
🔗

URL: https://twitchtokengenerator.com/

2

Twitch にログインして認証する

「Bot Chat Token」をクリックすると Twitch の認証画面が開きます。

🔒 id.twitch.tv/oauth2/authorize...
Twitch
id.twitch.tv
TwitchTokenGenerator があなたのアカウントへのアクセスを要求しています
許可するスコープ:
✓ chat:read — チャットの読み取り
✓ chat:edit — チャットへの送信
「許可する（Authorize）」をクリック
⚠️

必ずゲームで使う Twitch アカウントでログインしてください。
配信しているアカウントと同じで構いません。

3

ACCESS TOKEN をコピーする

認証が完了するとトークンが表示されます。ACCESS TOKEN の欄の値をコピーします。

🔒 twitchtokengenerator.com
✓ トークン取得完了
ACCESS TOKEN
abcdef1234567890abcdef1234567890abcd
REFRESH TOKEN
（使用しません）
📋 ACCESS TOKEN の値をまるごとコピーしてください。
REFRESH TOKEN は使いません。
🔐

トークンは秘密情報です。他人に見せたり、SNSに投稿したりしないでください。万が一漏れた場合は同サイトで「Revoke」して無効化できます。

4

アプリの設定画面に入力する

ChatViewPlayGame を起動して「⚙ Settings」を開き、以下のように入力します。

ChatViewPlayGame — 設定
⚙ Settings
Twitch Channel Name:
あなたのチャンネル名（小文字）
OAuth Token (oauth:xxxxx):
oauth:●●●●●●●●●●●●●●●●●●●●●●●●●●
⚠️ トークンの先頭に必ず oauth: を付けてから入力してください。
例: oauth:abcdef1234567890...
✅

入力完了したら「💾 保存して戻る」をクリック。
トップメニューに「Ready ✅」と表示されれば設定完了です！

🎉

設定完了！

あとはゲームを選んでスタートするだけです。
OBS では「ウィンドウキャプチャ」→「ChatViewPlayGame」を選択すれば配信に映せます。

よくある質問

トークンはいつ期限切れになりますか？

アクセストークンは約60日で期限切れになります。期限切れになったら同じ手順でトークンを再取得して設定し直してください。

「接続エラー」と表示される場合は？

① トークンに oauth: が付いているか確認
② チャンネル名が小文字になっているか確認
③ トークンを再取得して再入力してみてください。

トークンが漏れてしまった場合は？

twitchtokengenerator.com にアクセスして「Revoke Your Access Token」セクションで無効化できます。その後、再取得して設定し直してください。
